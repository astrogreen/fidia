    Overview

    Data are accessible from ADC in a variety of ways. Users can browse through each astronomical object's data products available per survey, including visualizations, processing information and associated properties. Users are able to query properties of astronomical objects across multiple surveys, and view/download a table of results. The query tool also allows for bulk download of specific types of data products for the resultant sample.

    Read more about each data access tool below.

    Tool About Link
    Data Browser    		 Per survey, browse the available data products and meta-data for each object. Provides a quick-look at the data products for a particular object. Data products can be downloaded as multi-extension FITS files, with all relevant processing information contained within the FITS header. Go to Data Browser
    Query Builder (SQL)    		 Query the ADC database across all surveys, filtering on some parameter(s) and return available rows of objects. Sort by column, download table as CSV. If your query result contains object IDs, the interface allows users to select particular data products from any of the surveys hosted at ADC to bundle into a downloadable tarball. Go to Query Builder
    Schema Browser    		 The schema browser allows you to explore the types of data product each survey team has released, how each was derived (reduction pipelines, analysis method etc), and any associated meta data. Go to Schema Browser
    TAP Service    		 ADC provides query access to our databases via the table access protocol (TAP). This services enables users to POST queries to the database over HTTP and retrieve responses in a chosen format (CSV, VOTable). TOPCAT (Tool for OPerations on Catalogues And Tables), an interactive graphical viewer and editor for tabular data, is also a fantastic tool for TAP service interaction. Coming Soon.
    Data Explorer    		 A graphical interface allowing users to view imaging (where available) and overlaid source positions for the ADC-hosted surveys. Coming Soon.
    Position Search    		 View available data (survey-derived data products or obervation coverage) for some radius around RA, DEC coordinates. Upload a list of RA, DEC positions and cross match to all available survey objects, to within some radius. Coming Soon.
    API    		 ADC's REST API provides programmatic access to read (and for survey team members, write) data, returning structured responses for you to use in your own code. Submit an SQL query, access previous query results (with authentication), read the meta data for a data product, and more. Responses are available in the popular JSON format, as well as FITS and CSV (where appropriate). ADC plans to document and release its API soon. Coming Soon.
