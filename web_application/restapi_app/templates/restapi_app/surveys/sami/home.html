    Data Access

    Data are accessible from ADC in a variety of ways. Whether you want to browse the available data products for a single astronomical source, create samples of objects from multiple surveys, or download data directly to your machine, our tools will help you get started.

    View the Data Access summary page for an overview.

    Data Browser

    Browse the available {{response.data.survey|uppercase}} data products and meta-data on an object-by-object basis. Provides a quick-look at the data products for a particular object, and allows you to download individual data products in fits format.

    Query Builder

    Create a sample of objects from all the surveys hosted at ADC using the Query Builder tool (or write complicated queries in SQL), filtering on some parameter(s) and return available rows of objects. From the results, select available products from the {{response.data.survey|uppercase}} for bulk download. Revisit previous queries in your Query History.
