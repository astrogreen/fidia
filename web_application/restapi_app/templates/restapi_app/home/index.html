API

Introduction

The AAO Data Central API provides a simple RESTful interface with lightweight JSON-formatted responses to power the ADC website's many features, including queries, data retrieval, feature requests, profile management, authentication and email support.

Users are able to access public data via the API in one of three formats: JSON, FITS or CSV. Using session or token authentication, users are able execute queries, and both read and write private data (e.g., query history).

You can read the API documentation here, or browse the API endpoints using the menu on the left.
