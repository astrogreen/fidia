    All AAODC tools in one place.

    AAODC offers a variety of tools to interact with available survey data products.


    Data Browser

    Available for: {% for survey in response.data.samples %}{{survey|uppercase}} {% endfor %}

    Per survey, browse the available data products and meta-data for each object. Provides a quick-look at the data products for a particular object, and download functionality for individual data products in fits format.

    Query Builder

    Available for: {% for survey in response.data.samples %}{{survey|uppercase}} {% endfor %}

    Create a sample of objects from all surveys, filtering on some parameter(s) and return available rows of objects. Select available products for bulk download. Revisit previous queries in your Query History.

    Schema Browser

    Available for: {% for survey in response.data.samples %}{{survey|uppercase}} {% endfor %}

    The schema browser allows you to explore the types of data each survey team has released, how each was derived (reduction pipelines, analysis method etc), and any associated meta data.

    Download

    Available for: {% for survey in response.data.samples %}{{survey|uppercase}} Query History {% endfor %}

    Download individual data products per source, samples of objects (in csv format) or data products for a user-created sample. If you wish to re-download a previous collection of data, you can visit your Download History.
