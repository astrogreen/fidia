    AAODC uses some custom terminology to organise data. Read more about the data model or view frequently asked questions.

    Samples

    Astronomical Objects

    Data Releases

    A data release is marked by the symbol, and represents a collection of data products a survey team has made publicly available. Survey teams typically release new-and-improved data products with each release, but if you ran your analysis on a particular piece of data at a particular time, data releases allow you to view a specific time-stamped version of the survey products.

    The Data Browser will always show the most current survey data, but you can switch to view previous data releases using the Data Releases button once you have selected the survey.

    Branches

    Branches represent different analysis pipelines for a particular data type. For example, a galaxy may have a stellar mass value calculated using method X, and a value calculated using method Y. These properties both belong to the galaxy's stellar mass property, and are contained within branches. Switching to a branch allows you to view all subsequent meta data within that branch.

    If a property has multiple branches, it will be marked by a symbol.

    Versions

    AAODC allows for astronomical object properties to be versioned, allowing team members to update a sub-set of data products without issuing a full team relese.

    If a property has multiple versions, it will be marked by a symbol.
